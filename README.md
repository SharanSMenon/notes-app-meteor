# Meteor Notes app
I made this app from a course on udemy by Andrew Mead

This is built using Meteor.

Go to https://notes-meteor-ssm.herokuapp.com to see a live demo
## Features
1. Stores notes in MongoDB
2. Has mobile menu for mobile navigation
3. User authentication
4. Sorts notes by last updated
